# 问题二分析报告

## 1. 问题二概述

根据题目要求，问题二需要构建融合多维度特征的风险预警模型，模型需输出高血脂症低、中、高三级风险，明确三级风险分别对应的特征分层阈值选取依据，并识别痰湿体质高风险人群的核心特征组合。

## 2. 输入输出分析

### 2.1 输入数据

从附件1提供的数据维度来看，输入数据包括：

| 维度 | 具体指标 |
|------|----------|
| 中医体质分型 | 九种体质标签及积分（1-9，积分0-100） |
| 中老年人活动能力 | ADL总分（0-50）、IADL总分（0-50）、活动量表总分（0-100） |
| 血常规 | 核心血脂指标（TC、TG、LDL-C、HDL-C）、关联代谢指标（血糖、血尿酸、BMI） |
| 基础信息 | 年龄组（1-5）、性别（0-1）、吸烟史（0-1）、饮酒史（0-1） |

### 2.2 输出结果

模型需要输出：
1. 高血脂症低、中、高三级风险分类
2. 三级风险分别对应的特征分层阈值选取依据
3. 痰湿体质高风险人群的核心特征组合

## 3. 建模公式与流程

### 3.1 特征工程

1. **特征提取**：
   - 原始特征：直接使用输入数据中的各项指标
   - 衍生特征：计算血脂比值指标（如TC/HDL、LDL/HDL、TG/HDL、AIP等）
   - 交互特征：体质与活动能力的交互项

2. **特征选择**：
   - 使用相关性分析、互信息、特征重要性等方法筛选关键特征
   - 重点关注能有效表征痰湿体质严重程度的指标

### 3.2 风险预警模型构建

#### 3.2.1 模型选择

考虑使用以下模型：
- **集成学习模型**（如随机森林、XGBoost）：能够处理非线性关系，对特征重要性有较好的评估
- **逻辑回归**：可解释性强，适合生成风险评分
- **支持向量机**：在小样本情况下表现良好

#### 3.2.2 模型训练与验证

1. **数据划分**：训练集（70%）、验证集（15%）、测试集（15%）
2. **模型训练**：使用训练集训练模型
3. **模型验证**：使用验证集调参，评估模型性能
4. **模型评估**：使用测试集评估模型的准确率、精确率、召回率、F1值等指标

### 3.3 风险分级阈值确定

采用以下方法确定风险分级阈值：

1. **基于模型输出概率**：
   - 低风险：模型预测概率 < 0.3
   - 中风险：0.3 ≤ 模型预测概率 < 0.7
   - 高风险：模型预测概率 ≥ 0.7

2. **基于特征组合**：
   - 低风险：血脂指标正常且痰湿积分 < 60，活动能力评分 ≥ 40
   - 中风险：血脂指标轻度异常或痰湿积分 60-79，活动能力评分 20-39
   - 高风险：血脂指标异常且痰湿积分 ≥ 60，或血脂正常但痰湿积分 ≥ 80 且活动能力评分 < 40

### 3.4 核心特征组合识别

使用以下方法识别核心特征组合：

1. **特征重要性分析**：通过模型输出的特征重要性排序
2. **关联规则挖掘**：发现特征间的关联关系
3. **聚类分析**：将具有相似特征的样本聚类，分析各类别的特征模式

## 5. 题目条件符合情况

### 5.1 符合的条件

1. **多维度特征融合**：模型融合了中医体质、活动能力、血常规等多个维度的特征
2. **三级风险输出**：模型明确输出低、中、高三级风险
3. **阈值选取依据**：提供了基于模型概率和特征组合的阈值选取方法
4. **核心特征组合识别**：使用多种方法识别核心特征组合

### 5.2 具体阈值依据

- **高风险**：符合题目中提到的"高风险可参考血脂指标异常且痰湿积分≥60，或血脂正常但痰湿积分≥80且活动能力评分低于40"
- **中风险**：介于低风险和高风险之间的特征组合
- **低风险**：血脂指标正常，痰湿积分较低，活动能力良好

## 6. 模型优势与创新点

1. **多维度融合**：综合考虑中医体质、活动能力和西医血脂指标
2. **可解释性**：不仅提供风险预测，还给出具体的特征阈值依据
3. **个性化风险评估**：针对不同特征组合的人群提供个性化的风险评估
4. **中西医结合**：将中医体质理论与西医检测指标相结合

## 7. 实施步骤

1. **数据预处理**：清洗数据，处理缺失值，标准化特征
2. **特征工程**：提取衍生特征，选择关键特征
3. **模型训练**：使用训练集训练多个模型，选择最优模型
4. **风险分级**：确定风险分级阈值，生成风险评估结果
5. **核心特征组合识别**：分析模型输出，识别核心特征组合
6. **模型验证**：使用测试集验证模型性能
7. **结果解释**：生成风险评估报告，解释核心特征组合

1. **风险预警模型**：能够预测高血脂症风险等级
2. **风险分级阈值表**：明确低、中、高风险的特征阈值
3. **核心特征组合**：识别出的痰湿体质高风险人群核心特征组合
4. **模型性能评估**：准确率、精确率、召回率等指标
5. **风险预警报告**：针对不同特征组合的风险评估结果
