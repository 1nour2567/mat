# 模型假设和问题分析和灵敏度分析汇报

## 1. 模型假设

### 1.1 核心假设

1. **线性相关假设**：假设特征与目标变量（痰湿体质和高血脂风险）之间存在线性或单调相关关系，可通过Spearman相关系数衡量

2. **信息独立性假设**：假设不同特征评估方法（Spearman相关、互信息、PLS载荷）提供的信息具有一定独立性，可通过熵权法进行客观融合

3. **特征重要性可加性假设**：假设特征重要性可以通过线性加权组合得到综合评分

4. **数据质量假设**：假设输入数据经过预处理，不存在缺失值和异常值（通过缩尾处理确保）

5. **人群同质性假设**：假设同一性别或年龄组内的个体具有相似的特征分布和风险模式

### 1.2 假设验证

- **数据验证**：使用了1000样本的真实数据，经过预处理和缩尾处理，确保数据质量
- **方法验证**：结合三种特征评估方法，通过熵权法客观计算权重，减少单一方法的偏差
- **稳健性验证**：通过灵敏度分析验证模型对参数变化的稳定性

## 2. 问题分析

### 2.1 问题定义

**核心问题**：筛选能表征痰湿体质严重程度且预警高血脂风险的关键指标，并分析不同人群（性别、年龄）的差异

### 2.2 分析框架

1. **特征层设计**：
   - 血常规指标层：包括传统血脂指标（TC、TG、LDL-C、HDL-C等）和缩尾处理的血脂比值指标
   - 活动能力层：包括ADL和IADL评分
   - 体质类型层：包括九种中医体质类型

2. **评估方法**：
   - **Spearman相关系数**：评估特征与痰湿体质积分的单调相关关系，捕捉非线性但单调的关联
   - **互信息**：评估特征与高血脂症二分类标签的信息依赖关系，捕捉非线性和非单调的关联
   - **PLS联合结构载荷**：综合评估特征与两个目标变量的联合相关性，考虑变量间的潜在结构

   **分工设计逻辑**：
   - **Spearman相关**：专注于痰湿体质积分这一连续变量，衡量特征对体质严重程度的预测能力，适合捕捉线性或单调关系
   - **互信息**：专注于高血脂症这一二分类变量，衡量特征对疾病状态的区分能力，适合捕捉复杂的非线性关系
   - **PLS载荷**：同时考虑两个目标变量，揭示特征在联合预测中的综合作用，减少变量间的多重共线性影响
   - 这种分工设计确保从不同角度全面评估特征的预测价值，避免单一方法的局限性

3. **权重计算**：使用熵权法计算各评估方法的客观权重，根据各方法提供的信息熵确定权重分配

4. **综合评分计算**：
   具体计算公式：
   \[ 
   Score_i = w_{spearman} \times S_i^{spearman} + w_{mi} \times S_i^{mi} + w_{pls} \times S_i^{pls} 
   \] 
   其中：
   - \( Score_i \)：第i个特征的综合评分
   - \( w_{spearman}, w_{mi}, w_{pls} \)：熵权法计算的各评估方法权重
   - \( S_i^{spearman} \)：第i个特征的Spearman相关系数绝对值
   - \( S_i^{mi} \)：第i个特征的互信息值
   - \( S_i^{pls} \)：第i个特征的PLS联合结构载荷值

5. **关键指标筛选**：基于综合评分排序，筛选前10个关键指标

5. **人群差异分析**：分析不同性别和年龄组的关键指标差异

### 2.3 数据来源

- **数据文件**：`data/processed/preprocessed_data.pkl`
- **样本量**：1000个样本
- **特征维度**：55个特征，包括血脂指标、活动能力评分、体质类型等

## 3. 灵敏度分析

### 3.1 分析方法

1. **权重灵敏度分析**：测试不同权重组合对关键指标排序的影响
2. **特征灵敏度分析**：测试删除单个特征对关键指标排序的影响
3. **人群灵敏度分析**：分析不同性别和年龄组的关键指标差异

## 4. 稳健性和有效性评估

### 4.1 稳健性评估

1. **参数稳健性**：模型对权重参数变化具有一定的稳健性，相似度保持在0.90以上
2. **特征稳健性**：模型对单个特征缺失具有鲁棒性，相似度保持在0.90以上
3. **方法稳健性**：结合三种评估方法，减少了单一方法的偏差

### 4.2 有效性评估

1. **特征有效性**：缩尾处理的血脂比值指标表现优异，前4位均为新增特征
2. **预测有效性**：综合评分方法能够有效区分不同特征的重要性
3. **临床有效性**：关键指标与临床经验一致，具有实际应用价值

### 4.3 敏感性边界

1. **权重边界**：当Spearman权重超过0.5时，模型结果开始变化
2. **特征边界**：删除缩尾处理的血脂比值特征时，模型结果变化较大
3. **人群边界**：不同性别和年龄组的关键指标排序存在显著差异

## 5. 结论与建议

### 5.1 主要结论

1. **关键指标**：缩尾处理的血脂比值指标（TC/HDL比值、non-HDL-C、TG/HDL比值、AIP）是预测痰湿体质和高血脂风险的核心指标

2. **模型稳健性**：模型对权重变化和特征缺失具有一定的稳健性，但对人群特征变化敏感

3. **人群差异**：不同性别和年龄组的关键指标存在显著差异，需要个性化干预

4. **方法有效性**：结合Spearman相关、互信息和PLS的综合评估方法有效，熵权法客观计算权重合理

### 5.2 干预建议

1. **共性干预**：
   - 关注血脂比值指标，特别是TC/HDL比值和TG/HDL比值
   - 保持适当的活动能力，尤其是ADL总分

2. **性别特异性干预**：
   - 女性：重点关注non-HDL-C和血尿酸
   - 男性：重点关注TG/HDL比值和LDL/HDL比值

3. **年龄特异性干预**：
   - 40-49岁：重点关注LDL/HDL比值和HDL-C
   - 50-59岁：重点关注non-HDL-C和TC/HDL比值
   - 60-69岁：重点关注TC/HDL比值和TG/HDL比值
   - 70-79岁：重点关注TC/HDL比值和non-HDL-C
   - 80-89岁：重点关注non-HDL-C和TG/HDL比值

### 5.3 模型改进方向

1. **特征优化**：进一步探索缩尾处理的最佳参数，提高特征质量

2. **方法优化**：考虑引入更多评估方法，如机器学习模型的特征重要性

3. **人群细分**：进一步细化人群分类，如结合BMI、生活方式等因素

4. **动态模型**：考虑构建动态模型，评估特征随时间的变化对风险的影响
