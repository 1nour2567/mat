# 高血脂症风险预警模型报告与详解

## 1. 模型概述

### 1.1 模型名称
三层融合预警模型（Triple-Layer Fusion Early Warning Model）

### 1.2 模型目标
- 实现高血脂症的早期风险预警
- 融合西医临床指标与中医体质特征
- 提供分层风险评估，支持精准干预

### 1.3 技术架构
- **第一层**：临床规则层（西医金标准）
- **第二层**：统计模型层（LightGBM预测）
- **第三层**：中医功能层（专家规则修正）

## 2. 模型架构详解

### 2.1 第一层：临床规则层（Clinical Rule Layer）

**核心功能**：基于血脂异常项数识别临床确诊高风险

**实现逻辑**：
- 计算四项血脂指标的异常项数
- 血脂异常项数 ≥ 1 判定为临床确诊高风险

**血脂异常判定标准**：
| 指标 | 正常范围 | 异常判定 |
|------|---------|----------|
| TC（总胆固醇） | 3.1-6.2 mmol/L | >6.2 或 <3.1 |
| TG（甘油三酯） | 0.56-1.7 mmol/L | >1.7 或 <0.56 |
| LDL-C（低密度脂蛋白） | 2.07-3.1 mmol/L | >3.1 或 <2.07 |
| HDL-C（高密度脂蛋白） | 1.04-1.55 mmol/L | <1.04 或 >1.55 |

**代码实现**：[three_layer_architecture.py](file:///workspace/Project_mat/src/three_layer_architecture.py#L22-L70)

### 2.2 第二层：统计模型层（LightGBM Prediction Layer）

**核心功能**：基于中西医融合特征预测潜在风险概率

**特征隔离**：
- **屏蔽特征**：血脂指标（防止标签泄露）
- **使用特征**：中医体质、活动能力、人口学特征、其他代谢指标

**模型配置**：
- 算法：LightGBM分类器
- 交叉验证：5折分层交叉验证
- 处理类别不平衡：is_unbalance=True
- 学习率：0.05
- 最大迭代次数：1000

**模型性能**：
- 5折交叉验证AUC：0.9121
- 整体OOF AUC：0.8902

**代码实现**：[three_layer_architecture.py](file:///workspace/Project_mat/src/three_layer_architecture.py#L79-L158)

### 2.3 第三层：中医功能层（TCM Functional Layer）

**核心功能**：基于中医专家规则修正风险等级

**风险等级定义**：
- 高风险：模型预测概率 ≥ 0.60
- 中风险：模型预测概率 0.20-0.60
- 低风险：模型预测概率 < 0.20

**中医规则修正**：
- **升档逻辑**：痰湿质 ≥ 60 且活动能力 < 40 → 升为高风险(中医预警)
- **降档逻辑**：痰湿质 < 17 且活动能力 ≥ 60 → 降为低风险(中医支持)

## 3. 特征工程

### 3.1 特征分类

**西医特征**：
- 血脂指标：TC、TG、LDL-C、HDL-C
- 代谢指标：空腹血糖、血尿酸、BMI
- 活动能力：ADL总分、IADL总分、活动量表总分
- 人口学特征：年龄组、性别、吸烟史、饮酒史

**中医特征**：
- 九种体质：平和质、气虚质、阳虚质、阴虚质、痰湿质、湿热质、血瘀质、气郁质、特禀质

### 3.2 特征隔离机制

**构建隔离墙**：
- **屏蔽清单**：血脂指标及其衍生指标（如AIP、TC/HDL等）
- **模型特征**：仅使用非血脂相关特征进行模型训练
- **预测逻辑**：血脂指标仅用于临床规则层判定

**防止标签泄露**：
- 血脂指标直接关联高血脂症标签，若用于模型训练会导致过拟合
- 隔离机制确保模型学习的是潜在风险因素，而非直接标签

## 4. 模型训练与预测流程

### 4.1 训练流程

1. **数据预处理**：加载并检查数据完整性
2. **临床规则应用**：计算血脂异常项数，标记临床确诊高风险
3. **模型训练**：使用5折交叉验证训练LightGBM模型
4. **模型评估**：计算AUC等性能指标
5. **模型保存**：保存训练好的模型用于预测

### 4.2 预测流程

1. **输入数据**：获取样本特征数据
2. **临床规则层**：检查血脂异常项数，判定临床确诊高风险
3. **统计模型层**：预测潜在风险概率
4. **中医功能层**：应用中医规则修正风险等级
5. **输出结果**：返回最终风险等级和预测概率

### 5.3 性能分析

- **优势**：
  - 临床规则层确保了高风险样本的准确识别
  - 统计模型层提供了潜在风险的量化评估
  - 中医功能层增加了个性化风险修正
  - 三层架构融合了中西医优势

- **局限性**：
  - 中风险样本稀少，风险等级分布不均衡
  - 低风险样本识别率较低
  - 模型依赖于特征质量和完整性

## 6. 技术亮点

### 6.1 三层融合架构

- **西医金标准**：确保临床准确性
- **机器学习模型**：提供量化风险评估
- **中医专家规则**：增加个性化修正
- **层层递进**：从客观指标到主观评估，全面覆盖风险因素

### 6.2 特征隔离机制

- **防止标签泄露**：确保模型学习的是因果关系而非直接关联
- **模拟临床场景**：模拟医生在不依赖血脂检测的情况下的风险评估
- **提升泛化能力**：模型能够识别潜在风险因素，而非仅依赖血脂指标

### 6.3 中医体质集成

- **创新融合**：将中医体质理论与现代机器学习相结合
- **个性化评估**：考虑个体体质差异对风险的影响
- **治未病理念**：通过中医体质特征实现早期预警

### 6.4 分层风险评估

- **三级风险**：高、中、低风险分层
- **精准干预**：不同风险等级对应不同干预策略
- **动态修正**：基于中医规则的动态风险调整
