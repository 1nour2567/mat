### 1.3 建模公式流程

#### 第一步：特征工程

```python
# 中西医交叉特征生成函数
def create_tcm_interactions(df):
    df['痰湿质×BMI'] = df['痰湿质'] × df['BMI']
    df['痰湿质×活动量表'] = df['痰湿质'] × df['活动量表总分']
    df['痰湿质×血尿酸'] = df['痰湿质'] × df['血尿酸']
    df['气虚质×BMI'] = df['气虚质'] × df['BMI']
    df['气虚质×活动量表'] = df['气虚质'] × df['活动量表总分']
    return df
```

#### 第二步：第一层 - 临床规则层（西医金标准）

**血脂异常判定：**
```
I_TC = 1, if TC > 6.2 或 TC < 3.1
I_TG = 1, if TG > 1.7 或 TG < 0.56
I_LDL = 1, if LDL-C > 3.1 或 LDL-C < 2.07
I_HDL = 1, if HDL-C < 1.04 或 HDL-C > 1.55
```

**血脂异常项数计算：**
```
N = I_TC + I_TG + I_LDL + I_HDL
```

**临床确诊高风险判定：**
```
临床确诊高风险 = 1, if N ≥ 1
临床确诊高风险 = 0, if N = 0
```

#### 第三步：第二层 - 统计模型层（LightGBM预测）

**模型设置（5折分层交叉验证）：**
```
LightGBM分类器参数：
  - objective = 'binary'
  - metric = 'auc'
  - is_unbalance = True
  - learning_rate = 0.05
  - n_estimators = 1000
```

**特征隔离机制（防止标签泄露）：**
```
LIPID_FEATURES = [
    HDL-C, LDL-C, TG, TC, AIP, TC/HDL比值, non-HDL-C,
    LDL/HDL比值, TG/HDL比值,
    non-HDL-C_缩尾, AIP_缩尾, TC/HDL比值_缩尾, LDL/HDL比值_缩尾, TG/HDL比值_缩尾,
    TC异常, TG异常, LDL-C异常, HDL-C异常, 血脂异常项数
]

MODEL_FEATURES = [所有非血脂相关特征 + 交叉特征]
```

**预测概率计算：**
```
p_hat = 1/k × Σ model_k.predict_prob(X), k=5
```

#### 第四步：第三层 - 中医功能层（边界修正）

**风险等级初步判定：**
```
初步风险等级 = 高风险, if p_hat ≥ 0.60
初步风险等级 = 中风险, if 0.20 ≤ p_hat < 0.60
初步风险等级 = 低风险, if p_hat < 0.20
```

**中医修正规则：**
```
# 升档规则（邪盛正衰）
if 初步风险等级 == 中风险 and 痰湿质 ≥ 60 and 活动量表总分 < 40:
    最终风险等级 = 高风险(中医预警)

# 降档规则（正盛邪微）
elif 初步风险等级 == 中风险 and 痰湿质 < 17 and 活动量表总分 ≥ 60:
    最终风险等级 = 低风险(中医支持)

else:
    最终风险等级 = 初步风险等级
```

**完整三层流程汇总：**
```
输入数据
    ↓
创建中西医交叉特征
    ↓
第一层：临床规则层
    ├─ 计算血脂异常项数 N
    └─ if N ≥ 1 → 临床确诊高风险 → 直接输出
    ↓（若 N = 0，继续）
第二层：统计模型层（无血脂特征）
    ├─ 训练 LightGBM（5折交叉验证）
    └─ 输出预测概率 p_hat
    ↓
第三层：中医功能层
    ├─ 初步判定风险等级
    ├─ 应用中医修正规则
    └─ 输出最终风险等级
```

#### 2.1 构建融合多维度特征的风险预警模型

#### 2.2 输出低、中、高三级风险

| 风险等级 | 判定条件 | 说明 |
|----------|---------|------|

#### 2.3 明确特征分层阈值选取依据

**三级风险对应的阈值选取依据：**

**高风险阈值依据：**

2. **统计模型依据：** 基于数据分布和验证，p_hat≥0.60为高概率
3. **中医修正依据：** 痰湿质≥60（显著偏颇）且活动能力<40（自理能力受限），符合题目示例中的“痰湿体质+低活动量”

**中风险阈值依据：**
- 统计模型预测概率0.20-0.60，数据驱动选取的不确定区间

**低风险阈值依据：**
- 统计模型预测概率<0.20，或中风险且痰湿质<17（体质平和）且活动能力≥60（活动能力强）

#### 2.4 识别痰湿体质高风险人群的核心特征组合

**核心特征组合（3大类）：**

**第1类：西医确诊组合（置信度100%）**
- 血脂异常≥1项（任意血脂指标超出题目附表1的正常范围）
- 支持度：89.1%（数据中血脂异常样本比例）

**第2类：中医预警组合（即使血脂正常）**
- 痰湿质≥60 + 活动量表总分<40
- 医学解释：痰湿体质显著偏颇且活动能力受限，即使血脂暂时正常，未来风险很高

**第3类：中西医融合交互特征组合**
- 痰湿质×BMI高（痰湿体质+肥胖）
- 痰湿质×活动量表低（痰湿体质+低活动）
- 痰湿质×血尿酸高（痰湿体质+代谢异常）
- 这些交叉特征表明中医体质与代谢指标的协同效应

#### 2.5 对结论给出合理的解释

**解释1：为什么选择三层架构？**
- 第一层（临床规则）保证西医金标准的准确性
- 第二层（统计模型）在无血脂信息的情况下挖掘潜在风险
- 第三层（中医修正）结合中医理论处理边界情况

**解释2：为什么严格隔离血脂特征？**
- 防止标签泄露：血脂特征与目标标签几乎一一对应
- 模拟真实临床场景：医生在不看血脂报告时也能判断风险
- 增强泛化能力：模型学习真正的生理关联而非直接关联

**解释3：为什么选择这些交叉特征？**
- 符合中医理论：痰湿体质与肥胖、活动能力、代谢的关系
- 安全可靠：不包含血脂相关项，无标签泄露风险
- 医学意义明确：可以解释为“痰湿体质加重了肥胖的风险效应”
