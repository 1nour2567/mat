## 2. 风险等级判定公式

### 2.1 临床规则层（西医金标准）

**血脂异常项数计算**：

\[ N_{lip} = \sum_{i=1}^{4} I_i \]

其中：
- \( I_1 = 1 \) 当 TC > 6.2 或 TC < 3.1，否则 0
- \( I_2 = 1 \) 当 TG > 1.7 或 TG < 0.56，否则 0
- \( I_3 = 1 \) 当 LDL-C > 3.1 或 LDL-C < 2.07，否则 0
- \( I_4 = 1 \) 当 HDL-C < 1.04 或 HDL-C > 1.55，否则 0

**临床确诊高风险判定**：

\[ Risk_{clinical} = \begin{cases} 
"临床确诊高风险", & N_{lip} \geq 1 \\
"非临床高风险", & N_{lip} = 0 
\end{cases} \]

### 2.2 统计模型层（LightGBM预测）

**特征隔离**：
- 屏蔽特征：血脂指标及其衍生指标
- 使用特征：中医体质、活动能力、人口学特征、其他代谢指标

**风险概率预测**：

\[ p_{hat} = \frac{1}{K} \sum_{k=1}^{K} LGBM_k(X) \]

其中：
- \( K = 5 \)（5折交叉验证）
- \( LGBM_k(X) \) 是第k折模型的预测概率
- \( X \) 是模型可用特征向量

### 2.3 中医功能层（专家规则修正）

**风险等级初始判定**：

\[ Risk_{initial} = \begin{cases} 
"高风险", & p_{hat} \geq 0.60 \\
"中风险", & 0.20 \leq p_{hat} < 0.60 \\
"低风险", & p_{hat} < 0.20 
\end{cases} \]

**中医规则修正**：

1. **升档逻辑**（邪盛正衰）：
   \[ 	ext{if } Risk_{initial} = "中风险" 	ext{ 且 } 	ext{痰湿质} \geq 60 	ext{ 且 } 	ext{活动能力} < 40 	ext{ 则 } Risk_{final} = "高风险(中医预警)" \]

2. **降档逻辑**（正盛邪微）：
   \[ 	ext{if } Risk_{initial} = "中风险" 	ext{ 且 } 	ext{痰湿质} < 17 	ext{ 且 } 	ext{活动能力} \geq 60 	ext{ 则 } Risk_{final} = "低风险(中医支持)" \]

**最终风险等级**：

\[ Risk_{final} = \begin{cases} 
"临床确诊高风险", & N_{lip} \geq 1 \\
"高风险(中医预警)", & 	ext{升档条件满足} \\
"低风险(中医支持)", & 	ext{降档条件满足} \\
Risk_{initial}, & 	ext{否则} 
\end{cases} \]

## 3. 特征分层阈值选取依据

### 3.1 高风险阈值
- **临床确诊高风险**：
  - 依据：血脂异常是高血脂症的直接表现，符合临床诊断标准
  - 阈值：血脂异常项数 ≥ 1

- **模型预测高风险**：
  - 依据：模型预测概率 ≥ 0.60，基于模型性能和临床经验
  - 阈值：p_hat ≥ 0.60

- **中医预警高风险**：
  - 依据：痰湿质≥60（痰湿体质严重）且活动能力<40（活动能力低）
  - 阈值：痰湿质≥60 且 活动能力<40

### 3.2 中风险阈值
- **依据**：模型预测概率在0.20-0.60之间，属于潜在风险区间
- **阈值**：0.20 ≤ p_hat < 0.60

### 3.3 低风险阈值
- **依据**：模型预测概率<0.20，风险较低
- **阈值**：p_hat < 0.20

- **中医支持低风险**：
  - 依据：痰湿质<17（痰湿体质轻微）且活动能力≥60（活动能力高）
  - 阈值：痰湿质<17 且 活动能力≥60

## 4. 痰湿体质高风险人群核心特征组合

### 4.1 核心特征组合识别

**组合一**：痰湿质≥60 + 低活动能力(<40) + 血脂异常
- **样本数**：24人 (16.6%)
- **风险等级**：临床确诊高风险

**组合二**：痰湿质≥60 + 低活动能力(<40) + 血脂正常
- **样本数**：2人 (1.4%)
- **风险等级**：中医预警高风险

**组合三**：痰湿质≥60 + 中等/高活动能力(≥40) + 血脂异常
- **样本数**：101人 (69.7%)
- **风险等级**：临床确诊高风险

### 4.2 特征组合分析公式

**风险贡献度计算**：

\[ RiskScore = w_1 \times 	ext{痰湿质得分} + w_2 \times (100 - 	ext{活动能力得分}) + w_3 \times N_{lip} \]

其中：
- \( w_1 = 0.5 \)（痰湿质权重）
- \( w_2 = 0.3 \)（活动能力权重）
- \( w_3 = 0.2 \)（血脂异常权重）

**高风险判定**：

\[ 	ext{if } RiskScore \geq 60 	ext{ 则为高风险} \]

## 5. 结论与解释

### 5.1 模型性能
- **AUC-ROC**：0.8902
- **准确率**：0.8820
- **风险等级分布**：临床确诊高风险91.1%，中风险0.2%，低风险8.7%

### 5.2 核心发现

1. **血脂异常是主要风险因素**：
   - 91.1%的样本存在至少一项血脂异常
   - TG异常最为常见（61.5%）

2. **痰湿体质与活动能力的重要性**：
   - 痰湿质≥60的人群中，86.2%存在血脂异常
   - 低活动能力显著增加高风险概率

3. **核心特征组合规律**：
   - 活动能力是活动强度的首要硬约束
   - 高频次（10次/周）是普遍最优选择
   - 高活动能力者可"以强度换频次"

4. **中西医融合的优势**：
   - 西医临床指标提供客观依据
   - 中医体质特征实现早期预警
   - 活动能力评估反映生活方式影响

### 5.3 临床意义

- **分层干预**：不同风险等级对应不同干预策略
- **个性化方案**：基于体质特征和活动能力制定干预计划
- **早期预警**：通过中医体质特征实现疾病前期的风险识别
- **综合管理**：结合西医指标和中医调理，实现全方位健康管理
