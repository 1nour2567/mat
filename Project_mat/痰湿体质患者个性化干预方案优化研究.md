
# 痰湿体质患者个性化干预方案优化研究

## 摘要

本文针对附件数据中体质标签取值为5的痰湿体质患者，结合中医调理原则与身体耐受度（活动干预强度），综合考虑经济成本及有效降低痰湿积分的目标，构建了优化模型，给出了不同患者特征的6个月干预方案，明确了"患者特征-最优方案"的匹配规律，并针对样本ID编号为1、2、3共计3位病患提供了详细的最优干预方案实现思路。

## 1. 优化模型构建

### 1.1 目标函数

本优化模型以**6个月后痰湿积分最低**为首要目标，以**总成本最低**为次要目标，以**干预强度和频次最低**为第三目标，构建了词典序优化函数。

### 1.2 约束条件

1. **预算约束**：单人6个月总成本≤2000元
2. **年龄约束**：
   - 80-89岁仅可选1级活动强度
   - 60-79岁可选1-2级活动强度
   - 40-59岁可选1-3级活动强度
3. **活动能力约束**：
   - 活动总分&lt;40仅可选1级活动强度
   - 40≤活动总分&lt;60可选1-2级活动强度
   - 活动总分≥60可选1-3级活动强度
4. **安全约束**：月总降幅≤20%
5. **活动干预起效约束**：每周活动频次≥5次

### 1.3 参数设置

| 中医调理级别 | 月成本（元） | 月降幅 |
|------------|-------------|--------|
| 1级（基础）| 30          | 1%     |
| 2级（标准）| 80          | 2%     |
| 3级（强化）| 130         | 3%     |

| 活动强度 | 单次成本（元） |
|---------|--------------|
| 1级（低）| 3            |
| 2级（中）| 5            |
| 3级（高）| 8            |

活动干预月降幅计算：当频次≥5次时，月降幅 = 3%×(强度-1) + 1%×(频次-5)

## 2. 患者特征-最优方案匹配规律

### 2.1 整体匹配规律

基于附件数据中278例痰湿体质患者的优化分析，得到以下核心匹配规律：

| 匹配规律编号 | 规律内容 | 说明 |
|------------|--------|------|
| 1 | 活动能力是活动强度的首要硬约束 | 活动总分&lt;40者仅能选择1级活动强度 |
| 2 | 年龄对活动强度的约束严格 | 60-79岁不可选3级，80-89岁仅可选1级 |
| 3 | 高频次（10次/周）是普遍最优选择 | 在预算允许时推荐最高频次 |
| 4 | 高活动能力者可"以强度换频次" | 活动能力高者采用3级强度配合8-9次/周也可达最优 |
| 5 | 3级中医调理是普遍最优选择 | 绝大多数患者选择3级强化调理 |

### 2.2 分特征匹配模式

#### 2.2.1 按活动能力分层

| 活动能力分层 | 最优方案模式 |
|------------|-----------|
| 低（&lt;40）| 3级中医 + 1级活动 + 10次/周 |
| 中（40-59）| 3级中医 + 2级活动 + 10次/周 |
| 高（≥60）| 灵活组合：可3级中医+3级活动+9次 或 3级中医+2级活动+10次 |

#### 2.2.2 按年龄组分层

| 年龄组 | 最优方案模式 |
|------|-----------|
| 40-49岁 | 3级中医 +（2-3级活动）+（9-10次/周）|
| 50-59岁 | 3级中医 +（1-2级活动）+（9-10次/周）|
| 60-79岁 | 3级中医 +（1-2级活动）+ 10次/周 |
| 80-89岁 | 3级中医 + 1级活动 + 10次/周 |

本研究构建了针对痰湿体质患者的个性化干预方案优化模型，结合了中医调理、活动干预、经济成本和身体耐受度等多维度因素。通过对278例痰湿体质患者的分析，明确了"患者特征-最优方案"的核心匹配规律。针对样本ID 1、2、3的具体分析表明，优化模型能够根据患者个体特征给出最适合的6个月干预方案，实现成本与效果的最优平衡。
