
# 高血脂风险预警模型 - 核心特征组合分析总结（V2.0）

## 1. 分析概述

本项目基于1000份样本数据，构建了融合中西医特征的三层风险预警模型（V2.0），并通过规则蒸馏和频繁项集挖掘方法，识别出高血脂高风险的核心特征组合。

**版本更新说明**：V2.0版本在原V1.0基础上扩展了特征工程，引入了尿酸异常标志和中医体质交叉特征，提升了模型的医学解释性和预测能力。

## 2. 整体数据概况

- **总样本数**：1000
- **高风险样本数**：891（89.1%）
- **血脂正常样本数**：109（10.9%）
- **模型性能**：OOF AUC = 0.8902

## 3. 核心特征组合识别结果

### 3.1 高频风险组合（含血脂异常）

由于在我们的数据集中，血脂异常样本通过临床规则直接被标记为高风险，因此以下组合的置信度均为100%：

| 核心特征组合 | 样本数 | 全体样本支持度 | 提升度 |
|------------|-------|-------------|-------|
| 活动能力低 + 血脂异常≥2项 | 96 | 9.6% | 1.12 |
| 痰湿质中 + 血尿酸高 + 血脂异常≥2项 | 93 | 9.3% | 1.12 |
| BMI正常 + 活动能力低 + 血脂异常≥2项 | 74 | 7.4% | 1.12 |
| 痰湿质高 + 血脂异常≥2项 | 60 | 6.0% | 1.12 |
| 痰湿质中 + 活动能力中 + 血尿酸高 + 血脂异常≥2项 | 63 | 6.3% | 1.12 |
| 气虚质低 + 活动能力低 + 血脂异常≥2项 | 63 | 6.3% | 1.12 |

### 3.2 血脂正常样本的潜在风险分析

在血脂正常的109个样本中：

- **痰湿质分布**：均值30.78，最高65，主要集中在低-中等水平
- **活动量表总分**：均值49.51，整体活动能力中等
- **BMI**：均值21.86，大部分在正常范围
- **血尿酸**：均值292.83，整体正常
- **模型预测概率**：最高0.20，整体风险较低

**血脂正常样本中常见的特征组合**：

| 特征组合 | 样本数 | 占比 |
|---------|-------|-----|
| 无高风险特征 | 40 | 36.7% |
| 痰湿质中 | 18 | 16.5% |
| BMI超重 | 9 | 8.3% |
| 痰湿质高 | 9 | 8.3% |
| 活动能力低 | 8 | 7.3% |
| 痰湿质中 + BMI超重 | 7 | 6.4% |
| 痰湿质高 + BMI超重 | 4 | 3.7% |

## 4. 风险特征阈值总结

### 4.1 西医特征阈值（临床标准）

| 指标 | 异常阈值 |
|-----|---------|
| TC（总胆固醇） | >6.2 或 <3.1 |
| TG（甘油三酯） | >1.7 或 <0.56 |
| LDL-C（低密度脂蛋白） | >3.1 或 <2.07 |
| HDL-C（高密度脂蛋白） | <1.04 |
| 血脂异常项数 | ≥1 项即为高风险 |

### 4.2 中医体质与活动能力阈值

| 特征 | 高风险阈值 | 说明 |
|-----|----------|-----|
| 痰湿质 | ≥60分 | 痰湿体质严重 |
| 活动量表总分（ADL+IADL） | <40分 | 活动能力低下 |
| BMI | ≥24 | 超重或肥胖 |
| 血尿酸 | >420 | 血尿酸偏高 |

## 5. 三级风险等级划分（修正后）

根据用户要求，我们调整了风险等级划分标准：

- **高风险**：
  - 血脂异常项数≥1（临床确诊）
  - 模型预测概率≥0.5
  - 模型预测概率在0.25-0.5之间且痰湿质≥60且活动能力<40

- **中风险**：
  - 模型预测概率在0.25-0.5之间，未触发升档规则

- **低风险**：
  - 模型预测概率<0.25
  - 模型预测概率在0.25-0.5之间且痰湿质<60且活动能力≥60

## 6. 临床干预建议

基于上述分析，针对不同风险等级的样本，建议如下干预措施：

### 6.1 高风险人群（临床确诊）

- 立即就医，接受正规降血脂治疗
- 严格控制饮食，减少高脂食物摄入
- 增加有氧运动，每周至少150分钟中等强度运动
- 痰湿质严重者建议中医调理

### 6.2 中风险人群

- 调整饮食习惯，控制饱和脂肪摄入
- 增加日常活动量，避免久坐不动
- 痰湿质者可进行中医体质调理
- 定期监测血脂水平

### 6.3 低风险人群

- 保持健康生活方式
- 定期体检，关注血脂变化
- 痰湿质或活动能力一般者可适当调理

## 7. 特征隔离措施（V2.0更新）

为避免标签泄露，模型训练时严格隔离了血脂相关特征：

### 7.1 被隔离特征（19个，V2.0扩展）

**原始血脂指标（4个）**：
- HDL-C（高密度脂蛋白胆固醇）
- LDL-C（低密度脂蛋白胆固醇）
- TG（甘油三酯）
- TC（总胆固醇）

**血脂衍生特征（14个，全部隔离）**：
- AIP（血浆致动脉粥样硬化指数）
- TC/HDL比值
- LDL/HDL比值
- TG/HDL比值
- non-HDL-C（非高密度脂蛋白胆固醇）
- 血脂异常项数
- 尿酸异常标志（仅在第二层隔离）

**交叉血脂特征（1个）**：
- 尿酸异常×性别

### 7.2 模型使用特征（25个，V2.0扩展）

**基础特征（20个）**：
- 中医体质（9个）：平和质、气虚质、阳虚质、阴虚质、痰湿质、湿热质、血瘀质、气郁质、特禀质
- 活动能力（3个）：ADL总分、IADL总分、活动量表总分（ADL+IADL）
- 人口学与生活方式（5个）：年龄组、性别、吸烟史、饮酒史、BMI
- 代谢指标（3个）：空腹血糖、血尿酸、尿酸异常标志

**新增中医交叉特征（5个，V2.0新增）**：
- 痰湿质×BMI
- 痰湿质×活动量表
- 痰湿质×血尿酸
- 气虚质×BMI
- 气虚质×活动量表

## 8. 总结（V2.0）

本分析成功识别出高血脂高风险的核心特征组合，突出了痰湿体质与活动能力对风险的重要影响。

**V2.0版本主要改进**：
1. **扩展特征隔离范围**：将隔离特征从7个扩展到19个，更全面地防止标签泄露
2. **新增尿酸异常标志**：引入安全的代谢异常指标，增强模型的医学解释性
3. **创建中医交叉特征**：新增5个体质×代谢指标的交叉特征，深化中西医融合
4. **提升模型容量**：从20个特征扩展到25个特征，在保证安全性的前提下提升预测能力

虽然在当前数据集中血脂正常样本中未发现典型的高风险组合，但通过模型预测和频繁项集分析，我们为未来更大规模数据分析提供了方法学基础。
