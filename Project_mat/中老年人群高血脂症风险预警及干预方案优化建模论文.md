# 中老年人群高血脂症的风险预警及干预方案优化

## 摘要

本研究针对中老年人群高血脂症的风险预警与干预方案优化问题，基于1000例病患个案数据，融合中医体质特征、活动能力评估与西医临床指标，构建了三层融合预警模型，实现了高血脂症的精准风险分层，并为痰湿体质患者设计了个性化干预方案。研究结果表明：（1）痰湿质、活动能力、BMI、血尿酸是表征痰湿体质严重程度和预警高血脂风险的关键指标；（2）三层融合预警模型（临床规则层+LightGBM预测层+中医功能层）具有良好的风险识别能力（AUC=0.8902）；（3）通过优化模型为不同特征的痰湿体质患者制定了个性化6个月干预方案，实现了成本与效果的最优平衡。本研究为中老年高血脂症的早期预警和精准干预提供了科学依据和实用工具。

## 关键词

高血脂症；风险预警；中医体质；活动能力；干预方案优化

## 1. 引言

### 1.1 研究背景

人口老龄化进程的持续加快，使得40岁及以上中老年人群成为慢性病防控的核心群体。高血脂症作为诱发冠心病、脑梗死等心脑血管疾病的重要危险因素，其发病率呈逐年攀升态势，严重威胁中老年群体的心血管健康与生活质量，也加重了基层医疗的慢病管理负担。

从中医体质学视角来看，体质禀赋与偏颇是疾病发生发展的内在根基，其中痰湿体质为中老年高血脂症的高发体质类型。痰湿体质人群存在痰湿内蕴、脾胃运化失常、水湿代谢不畅的核心特征，而脾失健运、痰湿阻滞恰与西医血脂代谢异常的病理机制高度契合，二者的病理关联为中西医结合防控高血脂症提供了重要理论支撑。

同时，中老年人日常活动能力的强弱直接影响机体气血津液的运行与脏腑功能的调和，与痰湿体质偏颇的改善、体内膏脂的运化代谢呈显著相关性，成为体质调理与血脂调控的重要外在干预抓手。

### 1.2 研究问题

目前临床中中老年高血脂症的筛查与风险评估仍单一依赖血常规血脂检测数据，缺乏对中医体质分型、日常活动能力等维度的综合考量，既无法对痰湿体质等偏颇体质人群实现未病先防的精准风险预警，也难以针对确诊患者结合体质特征制定个性化干预方案。

本研究旨在解决以下三个核心问题：

1. 从血常规体检指标、中老年人活动量表评分中，筛选出能有效表征痰湿体质严重程度、且能预警高血脂发病风险的关键指标；并研究九种体质对发病风险的贡献度差异。

2. 构建融合多维度特征的风险预警模型，模型需输出高血脂症低、中、高三级风险，明确三级风险分别对应的特征分层阈值选取依据，识别痰湿体质高风险人群的核心特征组合。

3. 针对确诊为“痰湿体质”的患者（附件数据中体质标签取值为5），结合中医调理原则与身体耐受度（活动干预强度），考虑经济成本及有效降低痰湿积分的目标下，构建优化模型，给出不同患者特征的6个月干预方案，明确“患者特征-最优方案”的匹配规律。

### 1.3 研究意义

本研究将中医体质理论与现代机器学习技术相结合，为中老年高血脂症的防控提供了新思路：

- 实现了中西医融合的风险预警模型，提高了风险识别的准确性和前瞻性
- 为痰湿体质患者提供了个性化的干预方案，体现了“治未病”的中医理念
- 为基层医疗的慢病管理提供了实用工具，有助于减轻医疗负担
- 丰富了慢性病防控的理论体系，促进了中西医结合的临床应用

## 2. 数据预处理与特征工程

### 2.1 数据概况

本研究使用附件1提供的1000例病患个案数据，包含以下维度：

- **中医体质分型**：九种体质标签及积分（0-100分）
- **中老年人活动能力**：ADL总分、IADL总分、活动量表总分（0-100分）
- **血常规指标**：核心血脂指标（TC、TG、LDL-C、HDL-C）、关联代谢指标（血糖、血尿酸、BMI）
- **高血脂症诊断信息**：二分类标签（0/1）、血脂异常分型标签（1/2/3）
- **基础信息**：年龄组、性别、吸烟史、饮酒史

### 2.2 数据清洗

数据清洗主要包括：

- 数据完整性检查：无缺失值
- 数据一致性检查：体质标签与积分一致性验证
- 数据规范性检查：确保各项指标符合临床正常范围

### 2.3 特征衍生

为了更好地表征血脂代谢状况，衍生了以下代谢指标：

- **non-HDL-C**：总胆固醇减去高密度脂蛋白胆固醇
- **AIP**（血浆致动脉粥样硬化指数）：log10(TG/HDL-C)
- **TC/HDL比值**：总胆固醇与高密度脂蛋白胆固醇的比值
- **LDL/HDL比值**：低密度脂蛋白胆固醇与高密度脂蛋白胆固醇的比值
- **TG/HDL比值**：甘油三酯与高密度脂蛋白胆固醇的比值

同时，为了减少极端值的影响，对上述代谢指标进行了1%-99%的缩尾处理。

### 2.4 特征隔离机制

为避免标签泄露，构建了特征隔离机制：

- **屏蔽特征**：血脂指标及其衍生指标（如AIP、TC/HDL等）
- **模型特征**：中医体质、活动能力、人口学特征、其他代谢指标
- **临床特征**：血脂指标仅用于临床规则层判定

## 4. 问题二：风险预警模型构建

### 4.1 模型架构

构建了三层融合预警模型（Triple-Layer Fusion Early Warning Model）：

#### 4.1.1 第一层：临床规则层（西医金标准）

**核心功能**：基于血脂异常项数识别临床确诊高风险

**实现逻辑**：
- 计算四项血脂指标的异常项数
- 血脂异常项数 ≥ 1 判定为临床确诊高风险

**血脂异常判定标准**：
| 指标 | 正常范围 | 异常判定 |
|------|---------|----------|
| TC（总胆固醇） | 3.1-6.2 mmol/L | >6.2 或 <3.1 |
| TG（甘油三酯） | 0.56-1.7 mmol/L | >1.7 或 <0.56 |
| LDL-C（低密度脂蛋白） | 2.07-3.1 mmol/L | >3.1 或 <2.07 |
| HDL-C（高密度脂蛋白） | 1.04-1.55 mmol/L | <1.04 或 >1.55 |

#### 4.1.2 第二层：统计模型层（LightGBM预测）

**核心功能**：基于中西医融合特征预测潜在风险概率

**模型配置**：
- 算法：LightGBM分类器
- 交叉验证：5折分层交叉验证
- 处理类别不平衡：is_unbalance=True
- 学习率：0.05
- 最大迭代次数：1000

**模型性能**：
- 5折交叉验证AUC：0.9121
- 整体OOF AUC：0.8902
- 准确率：0.8820

#### 4.1.3 第三层：中医功能层（专家规则修正）

**核心功能**：基于中医专家规则修正风险等级

**风险等级定义**：
- 高风险：模型预测概率 ≥ 0.60
- 中风险：模型预测概率 0.20-0.60
- 低风险：模型预测概率 < 0.20

**中医规则修正**：
- **升档逻辑**：痰湿质 ≥ 60 且活动能力 < 40 → 升为高风险(中医预警)
- **降档逻辑**：痰湿质 < 17 且活动能力 ≥ 60 → 降为低风险(中医支持)

## 5. 问题三：干预方案优化

### 5.1 优化模型构建

#### 5.1.1 目标函数

本优化模型以**6个月后痰湿积分最低**为首要目标，以**总成本最低**为次要目标，以**干预强度和频次最低**为第三目标，构建了词典序优化函数。

#### 5.1.2 约束条件

1. **预算约束**：单人6个月总成本≤2000元
2. **年龄约束**：
   - 80-89岁仅可选1级活动强度
   - 60-79岁可选1-2级活动强度
   - 40-59岁可选1-3级活动强度
3. **活动能力约束**：
   - 活动总分<40仅可选1级活动强度
   - 40≤活动总分<60可选1-2级活动强度
   - 活动总分≥60可选1-3级活动强度
4. **安全约束**：月总降幅≤20%
5. **活动干预起效约束**：每周活动频次≥5次

#### 5.1.3 参数设置

| 中医调理级别 | 月成本（元） | 月降幅 |
|------------|-------------|--------|
| 1级（基础）| 30          | 1%     |
| 2级（标准）| 80          | 2%     |
| 3级（强化）| 130         | 3%     |

| 活动强度 | 单次成本（元） |
|---------|--------------|
| 1级（低）| 3            |
| 2级（中）| 5            |
| 3级（高）| 8            |

活动干预月降幅计算：当频次≥5次时，月降幅 = 3%×(强度-1) + 1%×(频次-5)

### 5.2 患者特征-最优方案匹配规律

基于278例痰湿体质患者的优化分析，得到以下核心匹配规律：

| 匹配规律编号 | 规律内容 | 说明 |
|------------|--------|------|
| 1 | 活动能力是活动强度的首要硬约束 | 活动总分<40者仅能选择1级活动强度 |
| 2 | 年龄对活动强度的约束严格 | 60-79岁不可选3级，80-89岁仅可选1级 |
| 3 | 高频次（10次/周）是普遍最优选择 | 在预算允许时推荐最高频次 |
| 4 | 高活动能力者可"以强度换频次" | 活动能力高者采用3级强度配合8-9次/周也可达最优 |
| 5 | 3级中医调理是普遍最优选择 | 绝大多数患者选择3级强化调理 |

#### 5.2.1 按活动能力分层

| 活动能力分层 | 最优方案模式 |
|------------|-----------|
| 低（<40）| 3级中医 + 1级活动 + 10次/周 |
| 中（40-59）| 3级中医 + 2级活动 + 10次/周 |
| 高（≥60）| 灵活组合：可3级中医+3级活动+9次 或 3级中医+2级活动+10次 |

#### 5.2.2 按年龄组分层

| 年龄组 | 最优方案模式 |
|------|-----------|
| 40-49岁 | 3级中医 +（2-3级活动）+（9-10次/周）|
| 50-59岁 | 3级中医 +（1-2级活动）+（9-10次/周）|
| 60-79岁 | 3级中医 +（1-2级活动）+ 10次/周 |
| 80-89岁 | 3级中医 + 1级活动 + 10次/周 |

## 7. 结论与建议

### 7.1 主要结论

1. **关键指标识别**：痰湿质、活动能力、BMI、血尿酸是表征痰湿体质严重程度和预警高血脂风险的关键指标。痰湿质对发病风险的贡献度最高（0.182），其次是气虚质（0.145）和阳虚质（0.138）。

2. **风险预警模型**：三层融合预警模型（临床规则层+LightGBM预测层+中医功能层）具有良好的风险识别能力（AUC=0.8902），能够有效识别高风险样本，为临床决策提供支持。

3. **核心特征组合**：识别出了多个高风险特征组合，如"活动能力低+血脂异常≥2项"、"痰湿质中+血尿酸高+血脂异常≥2项"等，为风险评估提供了具体参考。

4. **干预方案优化**：构建了针对痰湿体质患者的个性化干预方案优化模型，明确了"患者特征-最优方案"的匹配规律，为不同特征的患者提供了科学、可行的6个月干预方案。

5. **样本分析**：针对样本ID 1、2、3的具体分析表明，优化模型能够根据患者个体特征给出最适合的干预方案，实现成本与效果的最优平衡。

### 7.2 建议

1. **临床应用建议**：
   - 推广三层融合预警模型，提高高血脂症的早期识别率
   - 重视痰湿体质的评估，将其作为高血脂症防控的重要参考指标
   - 加强活动能力评估，将其纳入常规健康检查

2. **干预策略建议**：
   - 针对不同活动能力和年龄组的患者，制定个性化的干预方案
   - 推广中医体质调理，特别是3级强化调理对痰湿体质的改善作用
   - 鼓励高频次、适合强度的活动干预，提高干预效果

3. **模型优化建议**：
   - 增加更多特征，如饮食、家族史等，提高模型的预测能力
   - 调整中风险阈值，使风险等级分布更加均衡
   - 考虑性别和年龄的差异化风险评估，提高模型的针对性

4. **数据建设建议**：
   - 增加更多样本，尤其是中风险样本，提高模型的泛化能力
   - 进行长期随访，验证模型预测的准确性和干预方案的效果
   - 建立多中心数据库，提高模型的适用性

### 7.3 展望

本研究为中老年高血脂症的风险预警和干预方案优化提供了新的思路和方法。未来的研究可以进一步：

- 探索更先进的机器学习算法，提高模型性能
- 开展临床验证研究，验证模型的实际应用效果
- 开发数字化工具，方便临床医生和患者使用
- 拓展研究范围，将模型应用于其他慢性病的防控
