# 问题三：痰湿体质患者个性化干预方案优化模型

## 1. 输入输出和具体建模公式流程

### 1.1 模型输入

**输入数据**（来自附件数据中体质标签取值为5的痰湿体质患者）：
- 痰湿积分：初始痰湿体质积分（0-100分）
- 年龄组：1=40-49, 2=50-59, 3=60-69, 4=70-79, 5=80-89
- 活动量表总分：ADL+IADL总分（0-100分）
- 其他特征：性别、BMI等（可能影响方案选择）

**干预措施参数**（来自题目附件）：
- 中医调理参数：各级调理的月成本和月降幅
- 活动干预参数：各级强度的单次成本和月降幅计算方式
- 预算约束：6个月总成本≤2000元

### 1.2 模型输出

**输出结果**：
- 最优中医调理级别：{1, 2, 3}
- 最优活动干预强度：{1, 2, 3}（受约束）
- 最优活动频次：{1, 2, ..., 10}
- 6个月后预期痰湿积分
- 6个月总成本
- "患者特征-最优方案"匹配规律

### 1.3 建模流程与公式

#### 1.3.1 变量定义

| 变量 | 含义 | 取值范围 |
|-----|------|---------|
|  t  | 中医调理级别 | {1, 2, 3} |
|  u  | 活动干预强度 | {1, 2, 3}（受约束） |
|  f  | 活动干预频次 | {1, 2, ..., 10} |
|  S₀  | 初始痰湿积分 | 实测值 |
|  S₆  | 6个月后痰湿积分 | 计算值 |
|  C  | 总成本 | 计算值 |

#### 1.3.2 约束条件

**活动强度约束集**：
```
U(age, activity) = {
    {1}                           年龄组=5 或 activity<40
    {1, 2}                       (年龄组∈{3,4} 且 activity≥40) 或 (年龄组≤2 且 40≤activity<60)
    {1, 2, 3}                  年龄组≤2 且 activity≥60
}
```

**其他约束**：
1. 预算约束：C ≤ 2000
2. 活动干预起效约束：f ≥ 5（为达到有效干预）
3. 安全约束：月降幅 ≤ 0.20

#### 1.3.3 成本计算模型

**月成本**：
```
C_monthly(t, u, f) = C_tcm(t) + 4 × f × C_activity(u)
```
其中：
- C_tcm(t) = {30, 80, 130} 为中医调理t级的月成本
- C_activity(u) = {3, 5, 8} 为活动强度u的单次成本
- 每月按4周计算

**总成本**：
```
C(t, u, f) = 6 × C_monthly(t, u, f)
```

#### 1.3.4 痰湿积分变化模型

**活动干预月降幅**：
```
r_activity(u, f) = {
    0.03×(u-1) + 0.01×(f-5)    f ≥ 5
    0                            f < 5
}
```

**总月降幅**：
```
r_monthly(t, u, f) = r_tcm(t) + r_activity(u, f)
```
其中 r_tcm(t) = {0.01, 0.02, 0.03} 为中医调理t级的月降幅

**6个月后痰湿积分**：
```
S₆ = S₀ × ∏_{i=1⁶} (1 - min(r_monthly(t, u, f), 0.20))
```

#### 1.3.5 词典序优化目标函数

目标函数采用词典序优化，优先级如下：

```
min_{t,u,f} [
    S₆(t, u, f),        # 首要目标：6个月后痰湿积分最低
    C(t, u, f),         # 次要目标：总成本最低
    (u + f)             # 第三目标：干预强度和频次最低
]

s.t.
    u ∈ U(age, activity)
    C(t, u, f) ≤ 2000
    f ≥ 5
    r_monthly(t, u, f) ≤ 0.20
```

#### 2.1 优化模型构建

我们构建了**词典序优化模型**：

基于对278例痰湿体质患者的优化分析，总结了核心匹配规律：
- 活动能力是活动强度的首要硬约束
- 年龄对活动强度的约束严格
- 高频次（10次/周）是普遍最优选择
- 高活动能力者可"以强度换频次"
- 3级中医调理是普遍最优选择
