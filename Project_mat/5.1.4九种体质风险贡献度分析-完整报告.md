## 二、第一部分：主标签边际风险模型

### 2.1 模型说明

设第i类主标签体质的人数为n_i，其中高血脂确诊人数为cases_i，则该体质组患病率定义为：
prevalence_i = cases_i / n_i

记总体患病率为prevalence_total，则第i类体质主标签的相对风险定义为：
relative_risk_i = prevalence_i / prevalence_total

- relative_risk_i > 1：该体质组患病风险高于总体平均水平
- relative_risk_i < 1：该体质组患病风险低于总体平均水平

### 2.2 卡方独立性检验

为检验"体质主标签"与"高血脂标签"是否存在显著统计关联，构造列联表并采用卡方独立性检验，统计量为：

χ² = Σ[(O_ij - E_ij)² / E_ij]

其中O_ij表示第i行第j列单元格中的观测频数，E_ij表示在独立性假设下的期望频数。

## 三、第二部分：连续积分净贡献模型

### 3.1 模型说明

主标签分析只能给出按组划分后的边际差异，难以刻画九种体质积分在控制其他变量后对发病风险的独立作用。为此，我们构建标准化Logistic回归模型。

### 3.2 数据标准化

对九种体质连续积分及控制变量进行Z-score标准化：

z_ij = (x_ij - μ_j) / σ_j

其中x_ij表示第i个样本在第j个体质维度上的原始积分，μ_j和σ_j分别表示该维度的样本均值与标准差。标准化处理后，各体质系数具有可比性。

### 3.3 Logistic回归模型

记y_i为第i个样本的高血脂标签，九种体质标准化积分向量为X_constitution_i，控制变量向量为X_control_i（包括年龄、性别、吸烟史、饮酒史等），则模型写为：

logit(P(y_i=1)) = α + Σ(β_j * X_constitution_i_j) + Σ(γ_k * X_control_i_k)

其中β_j表示第j个体质标准化积分的净贡献系数：
- β_j > 0：该体质维度升高会推动高血脂风险上升
- β_j < 0：该体质维度表现为相对保护方向

### 3.4 贡献度定义

为得到可比较的"贡献度"和"贡献率"，定义绝对贡献度为：
contribution_j = |β_j|

并进一步定义贡献率为：
contribution_rate_j = |β_j| / Σ(|β_j|)

其中contribution_j表示第j个体质维度的绝对净贡献强弱，contribution_rate_j表示该体质在九种体质净贡献总量中的占比。相比单纯比较β_j的大小，贡献率更适合回答"九种体质对发病风险的贡献度差异"这一题目要求。

## 五、总结与结论

### 5.1 主要发现

1. **主标签分析**：体质标签与高血脂风险在统计上无显著关联（卡方检验p=0.7004）
2. **连续积分分析**：
   - 平和质、气虚质、痰湿质、气郁质为风险升高体质
   - 血瘀质、湿热质、阳虚质、特禀质、阴虚质为保护方向体质
3. **贡献度排序**：
   - 平和质贡献最高（26.14%）
   - 气虚质次之（19.68%）
   - 血瘀质第三（16.58%）
4. **异质性分析**：不同性别和年龄组下，体质贡献模式存在差异

### 5.2 临床意义

本分析为理解中医体质与高血脂风险的关系提供了量化依据：
- 平和质和气虚质人群可能需要更关注高血脂预防
- 体质对风险的影响模式在不同性别和年龄段存在差异，需针对性干预
